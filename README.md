# Console File Manager (C++17)

Консольный файловый менеджер, реализованный на современном C++17 с использованием библиотеки `<filesystem>`.

## Возможности

- 📁 Навигация по файловой системе (вход в папки, возврат назад)
- 📋 Отображение содержимого директории с разделением на папки и файлы
- 🔤 Сортировка: папки сверху, файлы снизу, алфавитный порядок
- 📖 Чтение текстовых файлов с построчным выводом
- ⚠️ Обработка ошибок при работе с защищенными директориями

## Технологии

- C++17
- `std::filesystem` для работы с файловой системой
- `std::vector` и `std::sort` для хранения и сортировки данных
- Лямбда-выражения для кастомной сортировки
- Обработка исключений (`try-catch`)

## Как запустить

### Требования
- Компилятор с поддержкой C++17 (GCC 8+, Clang 7+, MSVC 2017+)
- CMake 3.10+ (опционально)

### Компиляция
```bash
g++ -std=c++17 main.cpp -o file_manager
./file_manager
